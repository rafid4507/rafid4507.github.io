Events and Programs

Hackathons AI and More

transform.10, brigk: Digitales Gründerzentrum Ingolstadt, THI, EDAG
09 Dec 2023 - 11 Dec 2023
Description
  • 48-hour event, 60 technicians, designers, computer scientists, creative minds and company representatives worked together in teams to develop innovative solutions to concrete problems
  • Were introduced to various challenges in the fields of Artificial Intelligence, Digitalization & Automation, Sustainable Mobility, Road Safety and Global Foresight
  • Our project builds a new interface for the VGI public transport app that enables users to use voice commands to interact with the app, making it more accessible to visually impaired users, and adding functionality to make it a better user experience.
  • The voice recognition can be triggered by the trigger word "Genie" and can take simple inputs regarding location and time and only in the English language. The text-to-speech prompts the user for each input when required and offers some advice regarding the weather which pulls live weather data for the city of Ingolstadt. The live bus tracking is currently simulated using sample data and can automatically refresh in the browser.

Singapore Autonomous Vehicle Challenge 2019

IEEE
07 Mar 2019 - 11 Mar 2019
Description
  • Task was to build an AUV(Autonomous Underwater Vehicle) that can perform given tasks, besides the simulations of tasks operational AUVs would have to be able to perform. The speed and accuracy at which the AUV performs, was the parameteres, used to decide the winner of the competition.
  • The tasks involve four widely faced challenges underwater such as AUV navigation, visual identification, acoustic localization and robotic manipulation.
  • The tasks were aimed to test the AUV’s acoustic and visual navigation capabilities, positioning, actuation and robotic manipulation, line following, image processing.
